Dumb Money Clips Poster

Privacy Policy

Last updated: July 20, 2026 · Home

Dumb Money Clips Poster ("the app") is a private tool operated by an individual to publish short highlight clips to their own social media accounts, including a connected TikTok account. It is not offered to the public as a service. This policy explains exactly what data the app touches and why.

In one sentence: the app stores a TikTok access token on the operator's own computer so it can upload the operator's own videos, and it reads back public view counts. Nothing is sold, and nothing is shared with third parties.

What the app accesses

  • A TikTok access token. When the operator authorizes the app through TikTok's official login, TikTok issues an access token. The app uses it only to upload videos to, and read basic metrics from, the operator's own connected account.
  • Video files the operator chooses to publish, and their captions.
  • Public metrics returned by TikTok for those posts (e.g. view counts), used only to measure the operator's own results.
  • Basic account info (user.info.basic) to confirm which account is connected.

What the app does NOT do

  • It does not collect data about any other TikTok users or the public.
  • It does not sell, rent, or share any data with third parties or advertisers.
  • It does not access private messages, contacts, or browsing activity.
  • It does not store passwords — authorization is via TikTok's token flow only.

Where data is stored

The access token and view counts are stored locally on the operator's own computer (in a local configuration file and database). They are not uploaded to any external server run by the app.

Data retention & deletion

The operator can revoke the app's access at any time from TikTok's connected-apps settings, which immediately invalidates the token. Deleting the local configuration file removes the stored token and metrics.

Children's privacy

The app is not directed to anyone under 18 and does not knowingly process data from minors.

Changes

If this policy changes, the "last updated" date above will change accordingly.
